# 数据库 SQL 说明

## 文件

| 文件 | 说明 |
|------|------|
| `volunteer_assistant_full.sql` | MySQL 8 完整导出（库结构 + 全量数据），数据库名 `volunteer_assistant` |

包含表：`user`、`school_info`、`major_info`、`admission_score`、`score_segment`、`volunteer_plan` 等共 17 张表。

## 导入步骤

1. 安装 MySQL 8，创建空库（可选，脚本内已含 `CREATE DATABASE`）：

```bash
mysql -uroot -p < volunteer_assistant_full.sql
```

Windows PowerShell 示例：

```powershell
& "C:\Program Files\MySQL\MySQL Server 8.0\bin\mysql.exe" -uroot -p < volunteer_assistant_full.sql
```

2. 在 `Backend/src/main/resources/application-local.yaml` 中配置与本机一致的 `spring.datasource` 用户名与密码。

3. 启动 Java 后端（`run-sql-on-startup` 建议保持 `false`，避免重复灌库）。

## 注意

- 导出时间以文件头注释为准；克隆后若需最新数据，请在本地重新执行 `mysqldump` 覆盖此文件。
- `user` 表含账号与密码哈希，**公开仓库请勿使用生产环境真实账号密码**；导入后建议修改默认测试账号密码。
