# 项目总览

## 1. 项目定位

**智选未来（Consultant）** 面向高考考生与家长，提供从「了解自我 → 查院校专业 → 量化测算 → 形成方案 → AI 辅助决策」的一站式志愿辅助。核心能力包括：

- 智能问答（RAG + 可选联网搜索）
- 全国院校 / 专业库与地图选校
- 冲稳保测算（Python 规则引擎 + MySQL 录取样本）
- 位次换算、分数线洞察、霍兰德兴趣测评
- 五步填报向导、志愿方案持久化与 AI 评审
- HTML 报告导出（Node 统一模板）

## 3. 模块职责

| 模块 | 技术 | 职责 |
|------|------|------|
| Backend | Java 17, Spring Boot 3.5, MyBatis-Plus | 用户认证、院校/专业/收藏、聊天流式、位次、向导、志愿方案、AI 评审、录取样本补全 |

## 5. 数据库

- 库名：`volunteer_assistant`
- 主要表：`user`、`school_info`、`major_info`、`admission_score`、`province_score_line`、`score_segment`、`volunteer_plan`、`volunteer_wizard_progress`、`chat_session` 等
- SQL 目录：`Backend/src/main/resources/sql/`
  - 新环境建议：`schema.sql` 或按需执行 `import_china_undergraduate_schools.sql`
  - 增量：`volunteer_plan.sql`、`score_segment.sql`（旧库升级）
- 启动开关（`application.yaml` → `consultant.data`）默认均为 **false**，避免每次启动自动灌库拖慢启动

## 6. 服务依赖关系

```
浏览器 :5173
    │ /api          → Java :8080  （主业务 + AI）
    │ /analytics    → Python :8001（冲稳保、洞察）
    └ /report-api   → Node :3001 （HTML 报告）

Java / Python  → MySQL
Java           → Redis（会话记忆、向量存储）
Java           → DeepSeek API（对话）
Java           → SiliconFlow Embedding + Tavily（可选联网）
```

未启动 Python/Node 时，院校库、问答、向导等仍可用；冲稳保页会提示「分析服务离线」。

## 7. 近期重要改动（维护参考）

| 项 | 说明 |
|----|------|
| 冲稳保样本补全 | `ensureMatcherPair`：同省只补本地；跨省只向目标生源省扩展，避免 28 省全量插入导致超时 |
| 启动性能 | 关闭 `StdOutImpl` SQL 日志；`run-sql-on-startup` / `seed-sample-admission-scores` 默认 false |
| 聊天预填 | 院校/专业/对比页跳转 `/chat?q=...` 会自动发送问题 |
| 导航 | 主栏 +「志愿工具」下拉 |
| 多服务 | 报告 HTML 统一在 Node `templates.js` |
